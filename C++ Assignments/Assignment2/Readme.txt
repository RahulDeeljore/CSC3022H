Type make to compile
Type make clean to remove .raw, .o and .data files
Type ./volimage MRI for default run
Type ./volimage MRI -g i (where i is an int) for the extracredit method
Type ./volimage MRI -x i j filename (where i & j are ints and filename is the name of the output file) for the extract method
Type ./volimage MRI -d i filename (where i is an int filename is the name of the output file) for the diffmap method


.raw files can be viewed using viewer.py
